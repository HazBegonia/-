📚 基于机器学习的图书推荐与检索系统 本系统是一个集自动化数据采集、全文检索、用户画像分析与智能推荐于一体的 Web 平台。项目采用前后端分离架构，通过爬取在线图书数据，利用机器学习算法实现精准的个性化书籍推荐。

🚀 核心功能 分布式数据爬取：基于 Scrapy 框架高效抓取图书多维信息（标题、价格、评分、简介等）。

智能推荐引擎：集成 XGBoost 预测模型，结合用户历史搜索与浏览轨迹，实现千人千面的图书推荐。

全文搜索与交互：基于 Django + Vue3 构建，支持快速书籍检索及收藏管理功能。

可视化数据管理：使用 MySQL 存储持久化数据，涵盖用户行为日志、图书元数据等核心维度。

🛠️ 技术栈 

爬虫端：Scrapy

后端：Django

前端：Vue3

机器学习：XGBoost + Scikit-learn + Pandas

数据库：MySQL
